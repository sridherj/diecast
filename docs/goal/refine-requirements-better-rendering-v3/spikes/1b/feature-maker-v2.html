Export Scheduler

Schedule a configured report to export on a recurring cadence so stakeholders receive fresh data automatically, with failures surfaced fast.

The job

Intent

When an analyst finishes configuring a report, they want to schedule it to export automatically on a recurring cadence, so that stakeholders receive fresh data without anyone re-running the export by hand.

How we'll know it works

SC-001

A schedule created with a valid cron expression produces its first export within one cadence interval.

SC-002

A failed export run delivers a failure notification to the schedule owner within five minutes.

SC-003

The export history view lists runs newest-first with a status badge per run.

Capabilities

FR-001

The scheduler shall accept a cron-style cadence expression and validate it before saving.

FR-002

The system shall run each due export within sixty seconds of its scheduled instant.

FR-004

The scheduler shall alert the report owner if a run fails.

FR-006

The scheduler shall let an operator pause and resume a schedule without deleting it.

Observability

FR-005

The system shall expose a read-only history of the last fifty export runs with their outcomes.

What people want to do

US1

Recurring schedule

As an analyst, I want to set a recurring cadence for a report export, so that stakeholders receive the report on schedule without anyone lifting a finger.

US2

Failure visibility

As an analyst, I want to be notified when a scheduled export fails, so that I can intervene before stakeholders notice missing data.

Guardrails

  • The export pipeline shall reuse the existing report-rendering service without modification.
  • Cadence expressions shall be limited to a minimum interval of fifteen minutes.

Not now

  • Ad-hoc one-off exports triggered manually outside any schedule.
  • Delivery channels other than email such as Slack or webhook in this version.

Directional ideas

  • A durable job queue such as a DB-backed outbox likely fits the at-least-once delivery need better than an in-process timer.
