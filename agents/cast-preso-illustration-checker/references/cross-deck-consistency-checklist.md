# Cross-Deck Consistency Checklist

> Run after every 3-4 approved illustrations. NOT per-illustration.

## Trigger

The orchestrator (Phase 5B) tracks illustration count and triggers this check.
Until Phase 5B is built, the HOW maker tracks count per invocation.

## Input Bounding

Compare each approved illustration since the last check against:
- The style anchor
- The 2 most recent previously-approved illustrations
- Total input: ~5-6 images per check

## Checks

| # | Check | Criteria |
|---|-------|----------|
| C.1 | Same medium feel | Same-type illustrations feel like the same artist. Group by type before comparing: watercolor-to-watercolor, SVG-to-SVG. SVGs are NOT drift from watercolors. |
| C.2 | Color palette match | Same 3-4 dominant colors as the style anchor |
| C.3 | Line quality match | Same brush weight, same edge treatment |
| C.4 | Mood consistency | Same warmth/playfulness (unless narrative demands contrast) |
| C.5 | Detail level consistency | No photorealistic piece in a loose-sketch deck |

## On Drift Detection

1. Flag the outlier illustration. Do NOT adjust the Style Bible.
2. Recommend regeneration with stronger style emphasis.
3. If drift persists after 2 corrections: accept and plan a post-processing color-grade pass.
