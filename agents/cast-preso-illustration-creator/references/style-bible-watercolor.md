# Watercolor Style Bible (Annie Ruygt)

> This is the FIXED style block for Slot 5 of every watercolor scene brief.
> Copy-paste VERBATIM. Never paraphrase, never "improve," never add synonyms.

## The Style Block

```
Hand-painted watercolor illustration with visible pencil underdrawing,
loose expressive brushwork on cold-pressed watercolor paper,
warm pastel palette of blush pink and dusty gold and sage green and warm cream,
soft diffused daylight, pigment bleed at edges, white paper showing through,
transparent watercolor washes with layered glazing, whimsical and gentle mood,
slightly quirky proportions, raw unpolished charm, children's book illustration warmth
```

## When to Override the Default

| Content Type | Use Instead | Rationale |
|--------------|-------------|-----------|
| Technical diagrams | SVG with deck color tokens | Precision required, animatable |
| Code showcases | HTML/CSS | Exact rendering needed |
| Data visualization | SVG with deck color tokens | Exact values, labels, proportions |
| Architecture diagrams | SVG with clean lines | Precision relationships |

When overriding: use CSS class names from the visual toolkit (`fill-accent`, `stroke-primary`, etc.) to maintain deck-level consistency.
